# Authors and Contributors

## Primary Authors

### Theoretical Foundation
**Sherwin Vakili**
- Developed the theoretical framework for the Iranian Calendar (IC) system
- Established the 3000 BCE epoch based on archaeological evidence
- Authored "On the Determination of the Epoch of History" (درباره تعیین مبدأ تاریخ)

### Implementation
**Alborz Teymoorzadeh**
- Python package implementation and architecture
- Algorithm development for calendar conversions
- Testing framework and validation
- Documentation and package distribution

## Attribution Requirements

Any use, modification, or derivative work of this calendar system must credit:
1. **Sherwin Vakili** - for the theoretical framework and historical research
2. **Alborz Teymoorzadeh** - for the software implementation
