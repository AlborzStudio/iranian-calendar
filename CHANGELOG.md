# Changelog

All notable changes to the Iranian Calendar project.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-25

### Added
- Initial release of Iranian Calendar
- Core date class with full validation
- 33-year cycle leap year algorithm
- Date conversions (IC ↔ Gregorian ↔ Solar Hijri)
- Date formatting (Persian, Latin, numeric, compact)
- Command-line interface
- Configurable calendar naming via config.py
- Comprehensive test suite (39 tests)
- Standalone single-file version
- Complete documentation
- MIT License

### Features
- Simplified Nowruz calculation (March 21 approximation)
- Multiple date format styles
- Date comparison operators
- Round-trip conversion validation
- Solar Hijri integration

### Documentation
- Complete README with usage examples
- Technical specification (SPEC.md)
- Theoretical foundation (ABSTRACT.md)
- Project plan with phased approach
- Usage examples and API reference
